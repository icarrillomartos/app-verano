# Carpeta `articulo`

Material para el artículo de LinkedIn sobre cómo nació **“La cuadrilla”**: una app para
que un grupo de amigos encuentre planes y se coordine, construida hablando con una IA.

## Contenido

- **`ARTICULO.md`** — el artículo redactado, en la voz de Iván, listo para publicar.
  Lleva marcadores `📸 FOTO n` donde van las imágenes.
- **`GUION-FOTOS.md`** — las 8 fotos que pide el artículo: qué capturar, cómo, dónde va
  cada una y su prioridad. (Las haces tú.)
- **`PROMPT.md`** — el prompt diseñado para escribir el artículo, con su justificación y
  cómo iterarlo.
- **`fotos/`** — carpeta vacía para que dejes aquí las capturas (nómbralas `foto-1.png`,
  `foto-2.png`, … como en el guion).
- **`Blog_OLD.docx`** — tus artículos anteriores; sirvieron de referencia de estilo.

## Cómo publicarlo

1. Lee `ARTICULO.md` y ajústalo a tu gusto (es tu voz, pero revísala).
2. Haz **público** el repo de GitHub (ver aviso al inicio de `GUION-FOTOS.md`).
3. Captura las 8 imágenes siguiendo `GUION-FOTOS.md` y déjalas en `fotos/`.
4. En el editor de artículos de LinkedIn, pega el texto y sustituye cada marcador
   `📸 FOTO n` por su imagen (borra la línea del marcador).
5. Portada sugerida del post: la FOTO 5 (app real) o la FOTO 3 (diseño).
